[HexoAdmin]([https://jaredforsyth.com/hexo-admin/]), 是一款非常优秀的 [hexo](http://hexo.io) 在线编写插件，支持所见即所得，直接生成预览效果。
使用也是非常简单，进入Blog的根目录

```
$ cd blog
```
安装插件
```
$ npm install hexo-admin
```
然后在浏览器中键入网址 http://localhost:4000/admin/ 就可以在线进行编辑了。
